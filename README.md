# Customize Agent

Customize Agent 是一个本地 AI 文档生成与知识库工作台。它帮助用户把自己的资料、提示词、角色配置、文档规范包和工作流模板组织起来，生成可校验、可追溯、可导出的业务文档。

## 它能做什么

- 管理本地知识库文件，区分“我的文件”和“内置示例”。
- 上传 PDF、Word、Excel、图片、Markdown、代码等资料并建立索引。
- 管理提示词，支持复制内置示例后按自己的业务修改。
- 配置文件角色、提示词角色和项目角色配置，让系统知道每份资料和每条提示词的用途。
- 创建文档规范包，定义事实字段、章节要求和可执行门禁规则。
- 使用工作流模板生成文档，并查看事实、来源、缺失项和校验结果。
- 导出 Markdown、HTML、PDF、Word 等格式。
- 管理生成资源、上下文和系统日志。

## 推荐使用流程

1. 在“模型配置”中添加可用模型，并检查连接状态。
2. 在“文件管理”中上传自己的资料，等待知识库索引完成。
3. 在“提示词管理”中创建提示词，或复制内置提示词后修改。
4. 在“角色配置”中创建文件角色、提示词角色，并组合成项目角色配置。
5. 在“文档规范包”中配置事实字段、章节规则和门禁规则。
6. 在“工作流模板库”中选择或创建模板，并绑定项目角色配置和规范包。
7. 开始生成文档，检查结构化事实、资料来源、缺失项和门禁结果。
8. 校验通过后导出正式文档。

## 我的数据与内置示例

系统会提供一些内置示例，帮助用户理解知识库、提示词、角色配置、规范包和模板之间的关系。

- “我的数据”是用户自己上传或创建的内容，默认优先展示。
- “内置示例”仅作为参考，不能直接覆盖或删除。
- 如果要使用内置示例，建议先复制为自己的版本，再根据业务修改。

## 本地数据说明

Customize Agent 默认把用户数据保存在本机，包括知识库索引、提示词、角色配置、规范包、模板、生成记录、生成资源和日志。用户可以围绕不同项目维护独立资料和生成结果。

## 常见问题

### 上传文件后为什么还在索引中？

文件上传完成后，系统会继续在后台写入知识库索引。请在文件管理页查看索引状态；如果失败，页面会展示失败原因，便于调整模型或资料后重试。

### 内置示例可以修改吗？

内置示例不能直接修改。点击编辑时系统会创建副本，用户可以修改副本并保存为自己的配置。

### 自定义门禁规则会生效吗？

会。文档规范包中的系统门禁类型和用户自定义门禁类型都会按配置参与生成校验和导出门禁。
